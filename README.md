# Post Writer

## Requirements

Currently it only works on Linux with x11.

**System Requirements**:
- `gedit`
- `xdotool`  - X11 automation tool

**Python Dependencies**:
- `pyautogui` GUI automation library

## Installation

You can simply install it with:
```bash
chmod +x setup.py
./setup.py
```
That will install `post_writer` in `/usr/bin/post_writer`.
